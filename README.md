# AIDUIKit

A  SwiftUI framework for creating dynamic, AI-driven UI components. AIDUIKit allows you to build components that can be serialized, transmitted, and rendered dynamically from JSON data.

## Project Structure

- **[`AIDUIKit/`](AIDUIKit/)** — The framework library (Swift Package). Contains the core component system, factory, and JSON serialization logic.
- **[`AIDUI/`](AIDUI/)** — Example app that demonstrates how to use AIDUIKit with sample chart components and a mock LLM service.
